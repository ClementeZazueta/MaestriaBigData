# MaestriaBigData

Repositorio principal para actividades y evidencias de la maestría.

## Frontend del portafolio

Se agregó una base profesional en `frontend/` con React + TypeScript + Vite para documentar:

- Materias cursadas
- Logros y enfoque profesional
- Trayectoria en formato blog/portafolio estático

Ver instrucciones de uso en [`frontend/README.md`](frontend/README.md).
