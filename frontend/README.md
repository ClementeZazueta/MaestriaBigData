# Frontend Portafolio (React + Vite)

Estructura base para un portafolio de maestría con contenido estático y enfoque profesional.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`

## Arquitectura

```text
src/
├─ app/            # composición principal
├─ components/     # bloques UI reutilizables
├─ data/           # contenido estático del portafolio
├─ features/       # secciones del sitio por dominio
├─ layouts/        # plantilla principal
├─ styles/         # estilos globales
└─ types/          # tipos compartidos
```
